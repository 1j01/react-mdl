## Demo

```jsx
{/* Simple spinner */}
<Spinner />

{/* Single color spinner */}
<Spinner singleColor />
```

#### Configuration

| Prop         | Type      | Effect       | Remarks      |
|:-------------|:----------|:-------------|:-------------|
| singleColor  | Boolean   | Use the primary color instead of a multi color spinner  | Optional |
